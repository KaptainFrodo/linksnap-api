Desktop — Chrome

Chrome Extension

Right-click any QR code image on any webpage to instantly decode and open the link. No screenshot needed — it works directly on the image in the page.

Download Extension
1
Download and unzip
Download the extension above and unzip it. manifest.json should be at the top level of the extracted folder — not inside a subfolder.
2
Open Chrome Extensions
In Chrome, go to chrome://extensions
3
Enable Developer Mode
Toggle Developer mode on using the switch in the top right corner.
4
Load the extension
Click Load unpacked and select the unzipped folder.
5
Use it
Right-click any QR code image on any webpage and select Open QR Link. The link opens in a new tab instantly.
Mobile — iOS

iOS Shortcut

A Shortcut that lives in your Control Center. Tap it while looking at any QR code on screen — it takes a silent screenshot, decodes it, opens the link, and deletes the screenshot automatically.

Install Shortcut
Tap Install Shortcut above on your iPhone. The Shortcuts app will open — tap Add Shortcut. Then follow the steps below to add it to Control Center.
1
Install the shortcut
Tap Install Shortcut above on your iPhone and tap Add Shortcut when the Shortcuts app opens.
2
Add to Control Center
Go to Settings → Control Centre. Tap + next to Shortcuts. Then tap the Shortcuts tile and select LinkSnap.
3
Use it
Find any QR code on your screen. Swipe down for Control Center and tap LinkSnap. The link opens in Safari automatically.
What happens when you tap it
Wait 0.5s — Control Center dismisses
Take Screenshot
Upload to LinkSnap API — ZBar decodes the QR
If a URL is found
Open in Safari
Otherwise
Show notification "No QR code found"
Delete screenshot — never saved to camera roll
iPhone 15 or later? Go to Settings → Action Button → Shortcut → select LinkSnap for one-button access without opening Control Center.

Backend

API

The iOS Shortcut uploads screenshots to a self-hosted API that decodes QR codes using ZBar — the same engine used in professional barcode scanners.

1
Decode endpoint
POST https://linksnap-api-8hn6.onrender.com/decode
Body: multipart/form-data with field image
{"success": true, "url": "https://example.com"}
2
Health check
GET https://linksnap-api-8hn6.onrender.com/ → {"status":"ok"}
